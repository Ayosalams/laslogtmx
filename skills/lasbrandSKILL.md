# laslogTMX Brand & Design Guidelines
Version: 2026-06-13

**Default Theme**: Light Modern (clean, professional, trustworthy for carriers/brokers)
**Dark Mode**: Toggle in user profile (Light is default)
**Accent Colors**: Electric Blue (#00bfff) + Neo-Red for CTAs

**Core Design Rules**:
- Rounded corners: rounded-3xl
- Subtle glassmorphism + soft blue glow on hover
- Military Time (24h) default everywhere
- Clean, professional, driver-friendly UI
- No flashy elements — focus on clarity and trust

**Key Features to Prioritize**:
- Company Team Chat + Load-specific Chat
- Military Time Display in header
- Receipt OCR with mandatory correction screen
- MOTUS Helper (high priority)
- FMCSA Compliance Hub
- Route Safety Widget
